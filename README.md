Opening Hours 營業時間表單與解析
---

## 說明

設定營業時間的 POC。

使用了 jQuery, Python/Flask。

程式碼很亂 :P


## 執行

先安裝 pip 套件：

```bash
virtualenv env
. env/bin/activate
pip install -r requires.txt
```

啟動

```bash
python openhour.py
```

接著訪問 [http://localhost:5000/](http://localhost:5000/)
